## Stack technique

- **Frontend Website** : Next.js 14, TypeScript, Tailwind CSS
- **Frontend App** : Next.js 14, TypeScript, Tailwind CSS, Supabase Auth
- **Backend** : Django 4, Django REST Framework, Python
- **Base de données** : MongoDB (données utilisateurs/emails) + SQLite (admin Django)
- **Hébergement** : serv00.net (production), GitHub (source)
